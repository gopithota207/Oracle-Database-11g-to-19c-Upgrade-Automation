Oracle 11g to 19c Upgrade Automation

Shell script to automate Oracle Database upgrade from 11.2.0.4 to 19c on Linux environments.

Features
Pre-upgrade validation checks
Oracle preupgrade utility execution
RMAN backup automation
Guaranteed Restore Point creation
Silent Oracle 19c software installation
Database upgrade using dbupgrade
Post-upgrade validation
Invalid object recompilation
Datapatch execution
Logging and email notifications
Requirements
Oracle Linux / RHEL
Oracle Database 11.2.0.4 source database
Oracle 19c software zip
Root or sudo access
